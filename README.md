# Kargo-ve-Teslimat-Y-netim-Otomasyonu
Bu otomasyon; e-ticaret lojistik süreçlerini optimize eden, kurye performansını takip eden ve iş günü bazlı dinamik teslimat hesabı yapan bir sistemdir. Üç hatalı denemede iade sürecini başlatır, gecikmiş siparişleri anlık listeler ve kurye verimliliğini grafiksel PDF raporlarına dönüştürür.
